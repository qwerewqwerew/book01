# 브랜치

{: .note }
> 브랜치란 : 소스의 버전
> 소스에 여러 버전을 추가 다양한 기능을 테스트 할수 있다
>

## 만들기
`git branch new-branch`

## 바꾸기
{: .note }
>브랜치를 만들었다고 해서 사용하고 있는것이 아니다. 빈 폴더만 만든 상태와 같다.
>브랜치를 바꿔야 새 브랜치에서 작업 할수있다.

`git switch new-branch`

## 만들면서바꾸기

`git switch -c new-branch`

## 지우기

`git branch -D new-branch`
